# نظام إدارة المدرسة الأهلية — v3.0

## التشغيل
```
pip install -r requirements.txt
python main.py
```

## المتطلبات
- Python 3.10+
- PySide6 للواجهة
- openpyxl لاستيراد Excel
- python-docx لاستيراد Word
- pytesseract + Pillow للمسح الضوئي (اختياري)

## التفعيل
**لا يوجد رمز تفعيل** — البرنامج يعمل مباشرة.

## أول تشغيل
1. أنشئ مدرسة جديدة
2. حدد اسم المستخدم وكلمة المرور لحساب المدير
3. ادخل وأضف باقي المستخدمين من "الإعدادات → المستخدمون"

## الصفوف المدعومة
- من الأول الابتدائي إلى السادس الابتدائي
- من الأول إلى الثالث المتوسط
- من الأول إلى الثالث الإعدادي (علمي + أدبي)

## استيراد البيانات
### Excel (xlsx):
| الاسم | سنة الميلاد | الجنس | الصف | الشعبة | الهاتف | العنوان | ولي الأمر | هاتف ولي الأمر | المهنة |

### Word (docx):
أي ملف Word يحتوي جدول بأعمدة تشمل: الاسم، الصف، الشعبة.
البرنامج يتعرف تلقائياً على أسماء الأعمدة.

### مسح السجلات الورقية:
اضغط "استيراد بالمسح" واختر صورة واضحة للسجل.
يتطلب تثبيت pytesseract وملف اللغة العربية.
